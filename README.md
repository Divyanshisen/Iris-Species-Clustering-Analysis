## Overview

This project was developed as part of the **Bee Skilled Internship Week 3** assignments.

The objective is to perform **K-Means Clustering** on the Iris dataset and visualize the clusters using **Principal Component Analysis (PCA)**. The project also compares the predicted clusters with the actual flower species.

---

## Dataset

- Iris Dataset (Iris.csv)

---

## Features Used

- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

---

## Machine Learning Algorithm

- K-Means Clustering (k = 3)

---

## Dimensionality Reduction

- Principal Component Analysis (PCA)

---

## Steps Performed

- Data Loading
- Data Preprocessing
- Removing the Id Column
- Feature Selection
- Applying PCA
- K-Means Clustering
- Cluster Visualization
- Comparison of Predicted Clusters with Actual Species
- Confusion Matrix Evaluation

---

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Output

The K-Means algorithm groups the Iris flowers into three clusters. PCA reduces the dataset to two principal components, making it easier to visualize the clusters. The predicted clusters are compared with the actual species to evaluate clustering performance.
